Er zijn maar 2 dingen die handmatig aangepast moeten worden om dit project aan het werk te krijgen:

- build.xml deploy locatie aanpassen
- alle jars in de /war/lib folder kopieren naar de tomcat lib folder

Om na het builden de website te openen, ga je naar de volgende link: localhost:8080/atd/

Hopelijk helpt dit om snel met ons project aan de slag te kunnen :)
